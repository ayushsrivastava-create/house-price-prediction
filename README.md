# House Price Prediction

This project predicts house prices using Machine Learning techniques.

## Dataset
California Housing Dataset

## Model Used
- Linear Regression

## Libraries Used
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Evaluation Metrics
- Mean Squared Error (MSE)
- R² Score

## Project Workflow
1. Data loading
2. Data preprocessing
3. Train-test split
4. Model training
5. Prediction
6. Model evaluation
